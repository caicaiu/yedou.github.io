# 简单句和复合句

## 简单句

简单句就是我们之前学的由五个基本句型构成的句子，包括它们的肯定句，疑问句和否定句。

1. 动作的句型（主语＋谓语＋宾语）
2. 状态的句型（主语＋系动词＋表语）
3. 也包括 there be句型以及被动句

## 复合句

又称主从复合句，由一个主句和一个或一个以上的从句构成，主句为句子的主体，从句不能独立，只用作句子的一个成分(表，状..)

从句: 

1. 连接词＋句子
2. 是复合句里面的一个成分。也就是“一个句子”作复合句的“主语，表语，宾语，状语，定语或同位语”
3. “从句”是不能单独出现的

一个句子在复合句里面充当了主语，就叫主语从句；充当了表语，就叫表语从句；充当了宾语，就叫宾语从句；充当了定语，就叫定语从句；充当了状语，就叫状语从句。

``主语从句，表语从句，宾语从句，同位语从句``这四个从句，英语称为“名词性从句”。

```
That I love you is known to all   主语从句
the truth is that I love you     表语从句
You know that I love  you    宾语从句
```



# 主语从句

主语从句就是在复合句里面作主语的那个句子。

主语从句本身可以是一个``主谓宾，主系表，there be或者被动句``的句子，当然，主语从句肯定也要有连接词

```python
That I love you
That Oscar is rich
That there is a book on the table
That the man is hit by the people
```

## 主系表

1. 主语从句＋系动词＋表语

	```python

	That he didn't want to come is obvious

	#（连接词 there be＋主语+（状语））＋系动词＋表语

	# （连接词 主语 + be + 过去分词）＋系动词＋表语
	```


## 造句步骤

从句有两个条件：

1. 必须是有连接词
2. 必须在复合句里面充当一个成分才能称为从句。

主语从句，表语从句，宾语从句，同位语从句的构成方式和意思都一样，连接词也相同。学会主语从句的构成方式，其他的三个从句的构成方式就不用学了。

### that构成的主语从句

连接词that是没有意思的

That引导主语从句的四种构成方式：

1. That＋主谓宾的句子

	```
	That we must learm English grammar first 
	```

2. That＋主系表的句子

	```
	That we are tired every day
	```

3. That＋there be 句型

	```
	That there are many people in the park 
	```

4. that＋被动句

	```
	That the boss was killed in the hotel
	```

以上仅仅是一个从句

主语从句＋系动词＋表语

```ABAP
That we must leam English grammar first is wise
That we are tired every day is true
That there are many people in the  park is strange
That the boss was killed in the hotel is true
```

主语从句＋谓语＋宾语  这种复合句句型不常见     主语从句一般表示一件事情，事情是很难发出一个动作的


### whether和if

whether和if有"是否"的意思，不担任句子成分，whether可以开头和结尾(it),if只能在结尾

1. whether/if+主谓宾的句子

	```
	Whether Jeff understands me.
	If you are listening to me carefully
	```

2. whether./if+主系表的句子

	```ABAP
	If Oscar is rich
	```

3. whether/if+there be句型

	```ABAP
	If there are a lot of trees in our village
	If there is a lot of gold in the island
	```

4. whether／if＋被动句

	```
	Whether/if he was killed last night 
	```

复合句    喜欢出现在“主语从句＋be＋表语”的复合句句型里面,注意if不能放在开头

```ABAP
It is doubtful whether／ if the explanation is reasonable  这个解释是否合理值得怀疑。
It is uncertain whether／ if we should invest thi s project我们是否应该投资这个项目还不确定
It doesn＇t matter whet her he com es or not他是否来都没关系的。
```

### when

有"什么时候的意思"，在从句里面充当时间状语，表语，可跟状态，动作的句子   

构成方式

1. When＋主谓宾＋地点状语 
	时态要求:只能用一般过去时，一般将来时态，偶尔用一般现在时态，不能用现在完成时态和现在进行时态
	不喜欢放后面

	```ABAP
	When you bought your house in Beijing 你什么时候在北京买房（一般过去时）
	When you will get married in Anhui 你什么时候在安徽结婚（一般将来时）
	When you do your homework 你什么时候做作业（经常性）（一般现在时）
	```

2. When＋主系表＋（状语）
	时态要求： 只能用在一般将来时态，一般过去时，一般现在时态，不能用在现在完成时态里面

	```ABAP
	when I will be famous（一般将来时）
	When Jeff was a teacher（一般过去时）
	When they will be back（一般将来时）
	```

3. When + there be 主语 +(状语) 规则跟上面一样

	```
	when there will be a school here（一般将来时态）
	```

4. when + be + 过去分词

	```
	when the boss is kill
	```

5. 充当表语  When＋主语＋系动词

	```
	When your birthday is 
	When the meeting is. 
	```

复合句

```
when there will be a school here is not known.
When we will be invited is not certain. 
```

### where

在主语从句中作状语，表语的作用，只有主谓宾从句  "哪里"

构成

1. 状语

	1. where + 主谓宾+ (状语)
		用在一般将来时态，一般过去时态，一般现在时态，现在进行时态，现在完成时态

		```
		Where you are going to learn English next year（一般将来时） 
		Where you drank beer last night（一般过去时） 
		```

2. 表语
	用在一般现在时态，一般过去时 一般将来，现在完成时态

	1. Where＋主语＋be＋状语 充当宾语从句比较多

		```
		Where my book is 
		Where she will be tomorrow 
		```

	2. 复合句

		```
		Where he has been（to） is still a mystery（where 在从句里面作表语）他曾经去过什么地方仍然是一个谜。
		Where s he was l ast night hasn＇t been known． （where在从句里面作表语）她昨天晚上在哪里还没有人知道
		
		```

复合句

1. where主语从句（主语）＋be＋表语

	```
	Where  you run every  moming is known to all 
	Where he  has bought  a house is a secret 
	Where she is hiding is not known 
	```

2. where主语从句（主语）＋be（助动词）＋及物动词过去分词（被动句）

	```
	Where  you are going to learn English next year is certain. 
	
	Where yo u drank beer  last night has not been known by your wife. 
	```

### why

Why在主语从句里面作原因状语，"为什么"

构成

1. Why＋主语＋谓语＋宾语

	```
	Why she will go to America next year
	```

2. Why＋主语＋系动词＋表语

	```
	Why he is rich 
	```

出现的复合句

1. why主语从句（主语）＋be＋表语

	```
	Why he is rich is a secret 
	```

2. why主语从句（主语）＋be（助动词）＋及物动词过去分词（被动句）

	```
	Why she is always sad makes me confused 为什么她老是伤心让我想不明白。（why引导的从句是状态的句子）
	```

### how

how在主语从句里面作方式状语。后面只可以跟动作的句子 ，"如何，怎样"

从句构成方式

1. How＋主语＋谓语＋（宾语）

	```
	1.How he opened the safe 
	2.How he escaped from the ja
	```

从句复合句

1. how主语从句（主语）＋be＋表语

	```
	How he  escaped from the jail is a mystery
	```

2. how主语从句（主语）＋be（助动词）＋及物动词过去分词（被动句）

	```
	How he  opened  the safe has not been known.
	```

# 表语从句

能做表语从句的也能做表语从句，意思就是不用学表语从句的构成了

在复合句中主语不能是人或物，大部分是抽象名词和指示代词

| fact 事实  | truth 真相 | question 问题 | problem 问题    | idea想法／主意 |
| ---------- | ---------- | ------------- | --------------- | -------------- |
| point 观点 | view 观点  | advice 建议   | suggestion 建议 | trouble 麻烦   |

主语（特定的词语）＋be（多数是is）＋表语从句

但是特定的词语喜欢跟不同的表语连接词

1. Fact，truth，idea，point，view，advice 和 suggestion 喜欢跟that

	```
	my point is that yo u can＇t finish t his work，我的观点是你不能完成这个工作。
	the fact is that the  girl never lo ved him．事实是这个女孩从未爱过他。
	```

2. question 和 problem 喜欢跟that以外的

	```
	The problem is who ca n hel p me  问题是谁可以帮我。
	the question is who can  go there  alone 问题是谁可以独自一个人去那里
	
	```

	

3. 指示代词this和that 喜欢跟that以外的，尤其是what

	```
	That is what I want 那就是我想要的。
	this is exactly what I  wanted t o say这就是我正想说的
	
	```

	



What主语从句＋be＋表语从句  超级复合句 没什么要求

```
What I want to know is where  you were last night ．我想知道的是你昨晚在哪里。

What I want to know is who com plained about m y course．我想知道的是谁抱怨我的课。
```
