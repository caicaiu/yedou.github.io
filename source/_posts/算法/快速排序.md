# 荷兰国旗问题

给定数组：[2, 3, 1, 9, 7, 6, 1, 4, 5]，给定一个值4，那么经过处理原数组可能得一种情况是：[2, 3, 1, 1, 4, 9, 7, 6, 5]

小于等于4的放左边，大于4的放右边

思想:

 从最右边找出一个目标值，数组最左边一个小于区域

1. 小于等于目标值，当前数和区域+1进行交换1 小于区往右扩，
2. 大于目标值，直接调下一个格

```
[2, 3, 1, 9, 7, 6, 1, 4, 5]
5为目标值 ，小于区域为-1
2 < 5 交换 向右扩1 移动一位  2
3 <5 交换  向右扩1  2,3
1 < 5 交换 向右扩1  2,3,1
9>5  不交换 直接下一个 2,3,1,9
7> 5 不交换 下一个
6>5 不交换
1<5 交换向右扩1  2,3,1,1),9,7,5
4<5 交换    2,3,1,1,4),9,7,5
```

思路2：  从最右边找出一个目标值，数组最左边一个小于区域,==放中间，右边有一个大于区域

1. 当前数小于目标值，小于区扩，然后下一个
2. 当前数== 目标值  直接跳下一个
3. 当前大于目标值，跟大于区域前一个交换，大于区域向左扩

```
[3,4,6,0,7,4,5,1,3]  目标值 4

```
