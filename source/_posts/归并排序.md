# 归并排序

归并排序是建立在归并（合并）操作上的一种有效的排序算法，归并排序对序列的元素进行逐层折半分组，然后从最小分组开始比较排序，合并成一个大的分组，逐层进行，最终所有的元素都是有序的

时间复杂度

归并排序算法每次将序列折半分组，共需要logn轮，因此归并排序算法的时间复杂度是O(nlogn)

空间复杂度

归并排序算法排序过程中需要额外的一个序列去存储排序后的结果，所占空间是n，因此空间复杂度为O(n)

稳定性

归并排序算法在排序过程中，相同元素的前后顺序并没有改变，所以归并排序是一种稳定排序算法

### 归并排序代码

递归实现

```java

public static void mergeSort1(int[] arr) {
    if (arr == null || arr.length < 2) {
        return;
    }
    //arr 1,3,2,4   
    process(arr, 0, arr.length - 1);
}

// 请把arr[L..R]排有序
// l...r 
// O(N * logN)
// arr 1,3,2,4  L=0  R = 3
public static void process(int[] arr, int L, int R) {
    if (L == R) { // base case
        return;
    }
    //中级值 (L+R)/2   mid = 1
    int mid = L + ((R - L) >> 1);
    //分割左边  0 ,1
    process(arr, L, mid);
    //分割右边  2,3
    process(arr, mid + 1, R);
    //合并   0,1,3
    merge(arr, L, mid, R);
}
//arr  1,3,2,4   L = 0   M = 1  R = 3   
public static void merge(int[] arr, int L, int M, int R) {
    //辅助数组   R-L+1     3-0+1 = 4
    int[] help = new int[R - L + 1];
    int i = 0;
    //左边初始
    int p1 = L;
    //右边初始
    int p2 = M + 1;
    //直到有一个越界
    while (p1 <= M && p2 <= R) {
        //那边小赋值给辅助数组
        help[i++] = arr[p1] <= arr[p2] ? arr[p1++] : arr[p2++];
    }
    // 要么p1越界了，要么p2越界了
    while (p1 <= M) {
        help[i++] = arr[p1++];
    }
    while (p2 <= R) {
        help[i++] = arr[p2++];
    }
    //还原给初始数组
    for (i = 0; i < help.length; i++) {
        arr[L + i] = help[i];
    }
}
```

```java
// 非递归方法实现 process
public static void mergeSort2(int[] arr) {
    if (arr == null || arr.length < 2) {
        return;
    }
    int N = arr.length;
    // 步长
    int mergeSize = 1;
    while (mergeSize < N) { // log N
        // 当前左组的，第一个位置
        int L = 0;
        //L是否超出边界
        while (L < N) {
            // M 就是左边的值 mid 
            int M = L + mergeSize - 1;
            //如果M超过长度，跳出
            if (M>=N) {
                break;
            }	
            // R值 最左边的值， 如果步长够的话 步长，不够则是最后一个
            int R = Math.min(M + mergeSize,N-1);
            merge(arr, L, M, R);
            //移动到下一个初始的位置
            L = R + 1;
        }
        // 防止溢出
        if (mergeSize > N / 2) {
            break;
        }

        //移动完后*2
        mergeSize <<= 1;
    }
}
```



# 归并排序的应用

### 最小和

返回数组中每个数字他左边数值比他小的数的和累加起来的总和

换个想法:每个数的右边比他大的数值的个数*当前数=> 每个比当前数大的的总和（当前数出现的个数）

```java
public static int process(int[] arr, int l, int r) {
    if (l == r) {
        return 0;
    }
    // l < r
    int mid = l + ((r - l) >> 1);
    return 
        //左边值
        process(arr, l, mid) 
        + 
        //左边的个数
        process(arr, mid + 1, r) 
        + 
        merge(arr, l, mid, r);
}

public static int merge(int[] arr, int L, int m, int r) {
    int[] help = new int[r - L + 1];
    int i = 0;
    int p1 = L;
    int p2 = m + 1;
    int res = 0;
    while (p1 <= m && p2 <= r) {
        //当前数值的右边有多少个比他大  r-p2+1 就是 左边的长-当前下标加1 
        res += arr[p1] < arr[p2] ? (r - p2 + 1) * arr[p1] : 0;
        //右边小拷贝右（产生小和），左边小拷贝不产生小和，等于先拷贝左边的
        help[i++] = arr[p1] < arr[p2] ? arr[p1++] : arr[p2++];
    }
    while (p1 <= m) {
        help[i++] = arr[p1++];
    }
    while (p2 <= r) {
        help[i++] = arr[p2++];
    }
    for (i = 0; i < help.length; i++) {
        arr[L + i] = help[i];
    }
    //多少数值的答案
    return res;
}
```

### 逆序对

思路: 从后面开始merge ,等于先拷贝右边，小于拷贝右边，大于拷贝左边

当右边比左边数值小的时候，计算右边数组的左边有多少个

![image-20221216134201802](https://xingqiu-tuchuang-1256524210.cos.ap-shanghai.myqcloud.com/4869/image-20221216134201802.png)

```java
// arr[L..R]既要排好序，也要求逆序对数量返回
// 所有merge时，产生的逆序对数量，累加，返回
// 左 排序 merge并产生逆序对数量
// 右 排序 merge并产生逆序对数量
public static int process(int[] arr, int l, int r) {
    if (l == r) {
        return 0;
    }
    // l < r
    int mid = l + ((r - l) >> 1);
    return process(arr, l, mid) + process(arr, mid + 1, r) + merge(arr, l, mid, r);
}

public static int merge(int[] arr, int L, int m, int r) {
    int[] help = new int[r - L + 1];
    int i = help.length - 1;
    int p1 = m;
    int p2 = r;
    int res = 0;
    while (p1 >= L && p2 > m) {
        //左边大于右边时，计算个数 p2 -m 也就是 当前所在位置-中间值
        res += arr[p1] > arr[p2] ? (p2 - m) : 0;
        //最后开始mege,右边小拷贝左边的
        help[i--] = arr[p1] > arr[p2] ? arr[p1--] : arr[p2--];
    }
    while (p1 >= L) {
        help[i--] = arr[p1--];
    }
    while (p2 > m) {
        help[i--] = arr[p2--];
    }
    for (i = 0; i < help.length; i++) {
        arr[L + i] = help[i];
    }
    return res;
}
```

### 数组中每个数大于右边数剩余2的个数

左边比右边小，右边不滑动，左边比右边\*2大，右边滑动，直到\*2后比等于或大于左边值，然后左边进行移动

继续划

![image-20221216135218337](https://xingqiu-tuchuang-1256524210.cos.ap-shanghai.myqcloud.com/4869/image-20221216135218337.png)

```java
public static int process(int[] arr, int l, int r) {
    if (l == r) {
        return 0;
    }
    // l < r
    int mid = l + ((r - l) >> 1);
    return process(arr, l, mid) + process(arr, mid + 1, r) + merge(arr, l, mid, r);
}

public static int merge(int[] arr, int L, int m, int r) {
    // [L....M] [M+1....R]
    int ans = 0;
    // 右边值
    int windowR = m + 1;
    //计算每组的的个数
    for (int i = L; i <= m; i++) {
        //如果 右边值不超当前数组，且左边的值大于右边的值剩余2
        while (windowR <= r && (long) arr[i] > (long) arr[windowR] * 2) {
            //滑动
            windowR++;
        }
        //结算总和 当前值- 右边值 - 1 = 左边的个数
        ans += windowR - m - 1;
    }
    int[] help = new int[r - L + 1];
    int i = 0;
    int p1 = L;
    int p2 = m + 1;
    while (p1 <= m && p2 <= r) {
        help[i++] = arr[p1] <= arr[p2] ? arr[p1++] : arr[p2++];
    }
    while (p1 <= m) {
        help[i++] = arr[p1++];
    }
    while (p2 <= r) {
        help[i++] = arr[p2++];
    }
    for (i = 0; i < help.length; i++) {
        arr[L + i] = help[i];
    }
    return ans;
}
```



### 数组 `nums` 两个整数 `lower` 和 `upper` 。求数组中，值在 `[lower, upper]` （包含 `lower` 和 `upper`）之内的 **区间和的个数** 。

题意: 数组中每个数值从往右相加的数值在[lower，upper]之间范围内的

```
[1,-1,2,3]  范围【-1,1】
0 -0 1 √ 在范围
0-1  [1,-1]  0 √
0-2  [1,-1,2] 2 × 不在范围
1-1  1-2 ... 
```

第一步: 前缀和

前缀和就是从位置1到位置i这个区间内的所有的数字之和。

```
原始数组  [-3,1,2,4,0,-1,5]
前缀后    [-3,-2,0,4,4,3,8]  每个位置前面的和
```

第二步：每个范围结尾前有多少个在这个范围内

```
[1,-1,2,3]
计算 每个位置之前有多少个在Lower,upper范围的
0-0  1 
1   0-1 1-1  两个范围
2   0-2 1-2 2-2 三个数
```

第三步: 小算法

```‘
sum[i-j] = sum[0-j]-sum[0-i-1];
所以
[0-17] 前缀和=100 原范围是 [10,40] 那么只要前面 0-5 在 60-90范围内，那么 6-17就是在该范围内
如 
0-0 40  范围[60-90]  0-0不在该范围内，所以 1-17 不在该 [10,,40]范围内
0-1 70  范围[60-90]  0-1在该范围内，所以 2-17 在 [10,40]范围内
```

第四部:简化题意

```
0-i 累加为x ,题意为 low,upper  以x结尾的子数组有多少个在[low,upper]范围上  参考第二步
		=
i之前所有前缀和中有多少个前缀和在 [x-upper,x-low]上
```

第五步: merge改写

```
前缀和:  [1,3,4,4,5]  [2,7,7,8,8,9]  原始范围在[0,5]上
第一步: 
2左边上有多少个在[-3,2]上的范围L,R指向初始值 L移动到大于等于-3的范围上，R移动到小于等于2的范围上  R-L 
第二步:
如果 merge时L == R 代表 0-R上的前缀和是否在 0,5范围上(没有比较了) 如果在返回1，不在返回0  
```

代码

```java
class Solution {
    public int countRangeSum(int[] nums, int lower, int upper) {
        //如果数组为空或者数组不存在返回0
        if (nums == null || nums.length == 0) {
            return 0;
        }
        //创建前缀和
        long[] sum = new long[nums.length];
        sum[0] = nums[0];
        for (int i = 1; i < nums.length; i++) {
            sum[i] = sum[i - 1] + nums[i];
        }
        //归并排序进行运算
        return process(sum, 0, sum.length - 1, lower, upper);
    }

    //分割元素
    public static int process(long[] sum, int L, int R, int lower, int upper) {
        //如果只包含本身的话，判断当前值是不是在指定的区间内，是返回1，不是返回0
        if (L == R) {
            return sum[L] >= lower && sum[L] <= upper ? 1 : 0;
        }
        //中间值
        int M = L + ((R - L) >> 1);
        //返回merge后的数值
        return process(sum, L, M, lower, upper) + process(sum, M + 1, R, lower, upper)
            + merge(sum, L, M, R, lower, upper);
    }

    //merge
    public static int merge(long[] arr, int L, int M, int R, int lower, int upper) {
        //计算变量
        int ans = 0;
        //左区间的数
        int windowL = L;
        //右区间的数
        int windowR = L;
        // [windowL, windowR)
        //遍历左区间的值
        for (int i = M + 1; i <= R; i++) {
            //最小值 
            long min = arr[i] - upper;
            //最大值
            long max = arr[i] - lower;
            //如果当前值大于边界值并且该值在最大的区间内，那么就进行移动
            while (windowR <= M && arr[windowR] <= max) {
                windowR++;
            }
            //如果当前值小于百年街并且该值的元素，小于最小值的haul就进行移动
            while (windowL <= M && arr[windowL] < min) {
                windowL++;
            }
            //相加返回
            ans += windowR - windowL;
        }
        //辅助数组，用于存储数据
        long[] help = new long[R - L + 1];
        //辅助数组的下标值
        int i = 0;
        //左边数组下标
        int p1 = L;
        //右边数组下标
        int p2 = M + 1;
        //判断大小
        while (p1 <= M && p2 <= R) {
            help[i++] = arr[p1] <= arr[p2] ? arr[p1++] : arr[p2++];
        }
        //代表左边值还有元素
        while (p1 <= M) {
            help[i++] = arr[p1++];
        }
        //代表有边值还有元素
        while (p2 <= R) {
            help[i++] = arr[p2++];
        }
        //辅助数组进行赋值
        for (i = 0; i < help.length; i++) {
            arr[L + i] = help[i];
        }
        return ans;
    }


}
```
