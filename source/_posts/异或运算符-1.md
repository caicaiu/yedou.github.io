# 异或运算 

 

异或运算：数字转换为位数，相同为0，不同为1 

# 运算定律 

XOR 运算有以下的运算定律。由于非常简单，这里就省略证明了。

1. 一个值与自身的运算，为0

	```java
	x ^ x = 0
	10
	10  
	00    
	```

2. 一个值与 0 的运算，总是等于其本身。

	```java
	x ^ 0 = x
	10
	00
	10 
	```

3. 可交换性

	```
	x ^ y = y ^ x
	10   11
	11   10
	01   01
	```

4. 结合性

	```
	x ^ (y ^ z) = (x ^ y) ^ z
	10 x         100 z
	 11  y       11 y
	100 z		 10 x
	111 z^y      01 x^y
	 10 x^(z^y) 100 (x^y)^z
	101         101
	```

## 提取出二进制最右侧的1来

```java
/**
 * 获取int 最后的值
 * @author 才
 */
public class InterviewTwo {
    public static void main(String[] args) {
        //获取而进制最后一个为1的
        System.out.println(anInt(12));
    }

    public static  int anInt(int target){
        /**
         * target&(-target) =   target&(~target+1)
   target&(~target+1)
   原始      0000100010101
   取反      1111011101010
   +1        1111011101011
   
   原始&(上面)      0000100010101  取出最左边的1
				   1111011101011 
				   0000000000001
         *
         * 首先target 是二进制源码，然后取二进制反码，然后取或
         *
         * & 就是如果相同为1,不相同为0
         */
        return target&(-target);
    }
}
```

## 两种数出现奇数次，其他数出现偶数次，找到并打印这两种数

```java
package day2.interview;

/**
 * 两个数时出现次数为奇数，其他数出现次数为偶数，返回出现次数为奇数的
 */
public class InterviewThere {
    public static void ant(int[] arr){
        int err = 0;
       
        for (int i = 0; i < arr.length; i++) {
            err ^= arr[i];
        }
        int onlyOne = 0;
        //取出最后一个1 a!= b err一定不为0 一定有一个二进制位 1(说明a和b的最后位的状态不一样的)
        int reverse =  err &(-err);
        for (int i = 0; i < arr.length; i++) {
            //取出状态不一样的数值不为1的取出来
            if((arr[i]&reverse)!=0){
                onlyOne^= arr[i];
            }
        }
        //结果
        System.out.println(onlyOne+" " +(onlyOne^err));
    }
}
```

## 有一种数出现K次，其他数出现M次M>1,K<M，找到出现K次的数

要求，额外空间复杂度0(1)，时间复杂度0N)

```java
package day2.interview;

import java.util.HashMap;
import java.util.HashSet;

public class InterviewFour {


    /**
     *
     * 思路：首先定义一个32位的数组(int字节的大小),从遍历数组中的元素，
     *      把该数组中每个二进制依次存入到该数组中,取模不等于0的那么就把1插入到相对应的位置
     *      然后进行返回即可
     */
    public  static int Km(int[] arr,int k,int m){
        int[] binaryTarget = new int[32];

        for (int i = 0; i < arr.length; i++) {
            for(int j = 0 ;j<31;j++){
//记录数组中的二进制位进入到数组中，arr[i]>>j&1 就是二进制向左移动j为再&1，该位置是0，就是0，是1就是1
                //比如：  000001010  j = 0  000001010 &1  移动0位在&后结果为0
                //                   j =1   00000101 &1 移动1位再&1结果为1，所以插入到里面去
                binaryTarget[j] += arr[i]>>j &1;
            }
        }
        int ant = 0;
        //遍历数组中每个数
        for (int i = 0; i < binaryTarget.length; i++) {
            /*//取模，如果不等于0的话，就把1插入到相对应的位置
            if(binaryTarget[i] % m != 0){
                //在第i个位置上插入
                ant|= (1<<i);
            }*/
            //出现的次数达不到k次
            if(binaryTarget[i]%m == 0){
                continue;
            }
            if(binaryTarget[i]%m==k){
                ant|= (1<<i);
            }else {
                return -1;
            }
        }
        //当ant为0的时候
        if(ant ==0) {
            int count = 0;
            for (int num : arr) {
                if (num == 0) {
                    count++;
                }
            }
            if (count!=k){
                return -1;
            }
        }
        return ant;
    }
}
```

## 剑指 Offer 56 - I. 数组中数字出现的次数

 

```java
class Solution {
    public int[] singleNumbers(int[] nums) {
        int err = 0;
        for(int i = 0;i<nums.length;i++){
            //两个奇数
            err^= nums[i];
        }
        int ans = 0 ;
        //最左边的1
        int reverse = err&(-err);
        for(int i = 0;i<nums.length;i++){
            //最左边不等于1的
            if((nums[i]&reverse)!=0){
                ans^= nums[i];
            }
        }
        return new int[]{ans,err^ans};
    }
}
```

class Solution {
    public int singleNumber(int[] nums) {
        int[] bit = new int[32];
        for(int i = 0;i<nums.length;i++){
            for(int j = 0;j<bit.length;j++){
                bit[j] += nums[i] >> j&1; 
            }
        }

                ans|= (1<<i);
            }
        }
        return ans;
    }
}
