# 链表

## 链表的分类

链表是物理存储单元上非连续的、非顺序的存储结构，数据元素的逻辑顺序是通过链表的指针地址实现，有一系列结点（地址）组成，结点可动态的生成。

结点包括两个部分：

1. 存储数据元素的数据域（内存空间）
2. 存储指向下一个结点地址的指针域。

相对于[线性表](https://so.csdn.net/so/search?q=线性表&spm=1001.2101.3001.7020)顺序结构，操作复杂。

链表分为 

1. 单链表
2. 双链表 
3. 单向循环链表 
4. 双向循环链表

```java
class Solution {
    //   a    ->   b    ->  c  -> a-> null  a
   
    public ListNode deleteNode(ListNode head, int val) {
       
      // head来到第一个不需要删的位置
		while (head != null) {
			if (head.val != val) {
				break;
			}
			head = head.next;
		}
		// 1 ) head == null    head = b->c->a->null
		// 2 ) head != null
		ListNode pre = head;  // pre = b ->c->a
		ListNode cur = head;  // cur = b ->c->a	
		while (cur != null) {      // true    true     true
			if (cur.val == val) {  // b !=a   c!=a     a == a
				pre.next = cur.next; 				  //b.next = null
			} else {
				pre = cur;       // pre = b   pre = c
			}
			cur = cur.next;     //cur = c     cur = a   cur = null
		}
		return head;  //head = b->c ->null
    }
}
```

# 栈和队列

栈（Stack）是限定在仅在表尾插入的[线性表](https://so.csdn.net/so/search?q=线性表&spm=1001.2101.3001.7020)。

队列：队列是一种 **先进先出** 的线性表。它值允许在表的一端进行插入，而在另一端进行删除元素

## 栈的应用

### 怎么用数组实现不超过固定大小的队列和栈？

```java
//数组实现队列
public class Code04_RingArray {

	public static class MyQueue {
		private int[] arr;
		private int pushi;// end
		private int polli;// begin
		private int size;
		private final int limit;

		public MyQueue(int limit) {
			arr = new int[limit];
			pushi = 0;
			polli = 0;
			size = 0;
			this.limit = limit;
		}

		public void push(int value) {
			if (size == limit) {
				throw new RuntimeException("队列满了，不能再加了");
			}
			size++;
			arr[pushi] = value;
			pushi = nextIndex(pushi);
		}

		public int pop() {
			if (size == 0) {
				throw new RuntimeException("队列空了，不能再拿了");
			}
			size--;
			int ans = arr[polli];
			polli = nextIndex(polli);
			return ans;
		}

		public boolean isEmpty() {
			return size == 0;
		}

		// 如果现在的下标是i，返回下一个位置
		private int nextIndex(int i) {
			return i < limit - 1 ? i + 1 : 0;
		}

	}

}
```



### 实现一个栈，在基本功能的基础上，再实现返回栈中最小元素的功能

```java
class MinStack {

    //定义两个栈
    Stack<Integer> stackStack;
    Stack<Integer> minStack; 

    /** initialize your data structure here. */
    public MinStack() {
        stackStack = new Stack<>();
        minStack = new Stack<>();
    }
    
    public void push(int x) {
        //判断最小栈是否为空
        if(minStack.isEmpty()){
            //存入栈中
            minStack.push(x);
        }else if(minStack.peek() >= x){//判断当前是否等于或者等于最小栈上面的值，如果等于或小于就压栈
            minStack.push(x);
        }
        //存入栈中
        stackStack.push(x);
    }
    
    public void pop() {
        //判断栈是否为空
        if(stackStack.isEmpty()){
            return;
        }
        //判断出栈的元素是否等于最小栈上面的值，如果是，最小栈出栈
        int val = stackStack.pop();
        if(val==minStack.peek()){
            minStack.pop();
        }
    }
    
    public int top() {
        if(stackStack.isEmpty()){
            return -1;
        }
        //返回栈顶元素，但是不出站
        return stackStack.peek();

    }
    
    public int getMin() {
        //判断最小栈是否为空
        if(minStack.isEmpty()){
            return -1;
        }
        return minStack.peek();
    }
}
```

### 如何用栈结构实现队列结构

```java
class MyQueue {

    Stack<Integer> popStack;  // 出栈
    Stack<Integer> pushStack; //入栈

    public MyQueue() {
        popStack = new Stack<>();
        pushStack = new Stack<>();
    }
    
    //编写方法，将入栈倒入出栈上
    public void pushPop(){
        //判断出栈是否为空，如果为空倒入，不为空，要消费完才能倒入
        if(popStack.isEmpty()){   //    true  false   
            while(!pushStack.isEmpty()){   // true  
                popStack.push(pushStack.pop()); //pop = 1  push = 2,3,4,5
            }
        }
    }
    public void push(int x) {
        //入栈，然后导入数据
        pushStack.push(x); //1 2 3 4 5 连续
        //导入 
        pushPop();
    }
    // 1 , 2 连续
    public int pop() {
        //判断两个栈是否为空
        if(popStack.isEmpty()&&pushStack.isEmpty()){ 
            return -1;
        }
        //导入栈
        pushPop();   //  pop = false  pop = 5,4,3,2
        //返回出栈数据 
        return popStack.pop();  // 1   2 3 4 5 
    }
    
    public int peek() {
        //判断栈是否为空
        if(popStack.isEmpty()&&pushStack.isEmpty()){ 
            return -1;
        }
        //导入栈
        pushPop();

        //返回栈顶数据
        return popStack.peek();
    }
    
    public boolean empty() {
         return popStack.isEmpty()&&pushStack.isEmpty();
    }
}

```

### 用队列结构实现栈结构

```java
class MyStack {

    //当前队列
    public Queue<Integer> queue;
	//辅助队列
    public Queue<Integer> help;


    public MyStack() {
        queue = new LinkedList<>();
		help = new LinkedList<>();
    }
    
    public void push(int x) {   // queue = 1,2,3,4,5
        //进入队列
        queue.offer(x);
    }
    
    //出栈
    public int pop() {
        //如果当前队列的个数大于1，那么就全部移动到辅助队列上
        while (queue.size() > 1) {   
			help.offer(queue.poll()); // queu = 1   help = 2,3,4,5
		}
        //出队
		int ans = queue.poll();  // 1
        //两队列交换，因为如果再次入栈的话，就要进入到queue中，这样才可以拿到栈顶元素
		Queue<Integer> tmp = queue;    // null
		queue = help; //que = 2,3,4,5
		help = tmp;   // help  = null
        //返回当前元素
		return ans;
    }
    
    //获取栈顶
    public int top() {
        //如果当前队列的个数是大于2的，那么就倒入到辅助栈中
        while (queue.size() > 1) {    
			help.offer(queue.poll()); // queu = 1   help = 2,3,4,5
		}
        //只剩下最后一个元素（栈顶）返回即可
		return queue.peek();  //queu = 1 help  = 2345
    }
    
    public boolean empty() {
        return queue.isEmpty();
    }
}
```

# 递归

递归的基本思想是某个函数直接或者间接地调用自身

### 例子

求数组arL.…R]中的最大值，怎么用递归方法实现。

1. 将[L.R]范围分成左右两半。左：[LMid右[Mid+1R]
2. 左部分求最大值，右部分求最大值
3. [L.R]范围上的最大值，是max{左部分最大值，右部分最大值}

注意：2)是个递归过程，当范围上只有一个数，就可以不用再递归了

```java
// 求arr中的最大值
public static int getMax(int[] arr) {
	return process(arr, 0, arr.length - 1);
}

// arr[L..R]范围上求最大值  L ... R   N
public static int process(int[] arr, int L, int R) {
	// arr[L..R]范围上只有一个数，直接返回，base case
	if (L == R) { 
		return arr[L];
	}
	// L...R 不只一个数
	// mid = (L + R) / 2
	int mid = L + ((R - L) >> 1); // 中点   	1
	int leftMax = process(arr, L, mid);
	int rightMax = process(arr, mid + 1, R);
	return Math.max(leftMax, rightMax);
}
```

## Master公式

形如
T(N)=a*T(N/b)+O(NAd)(其中的a、b、d都是常数)  a是要递归多少次，b是递归的
的递归函数，可以直接通过Master公式来确定时间复杂度
如果log(ba)<d,复杂度为O(Nnd)
如果log(b,a)>d,复杂度为O(Nlog(b,a)
如果log(b,a)==d,复杂度为O(NAd*logN)
